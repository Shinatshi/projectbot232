<h1>AI Career Finder</h1>

<p><strong>AI Career Finder</strong> — консольный бот на Python, который помогает пользователю подобрать подходящие профессии на основе его интересов и навыков.  
Бот использует искусственный интеллект и морфологический анализ, чтобы понимать смысловые запросы на русском языке и выдавать точные рекомендации.</p>

<hr>

<h2>✨ Основная идея</h2>

<p>
Цель проекта — сделать карьерное ориентирование простым и персонализированным.  
AI Career Finder анализирует запрос пользователя, сравнивает его с базой данных профессий и подбирает подходящие варианты с описанием и ключевыми навыками.
</p>

<ul>
  <li>Понимает естественный язык (в том числе синонимы и формы слов)</li>
  <li>Использует AI-модель <code>GPT4All</code> для генерации рекомендаций</li>
  <li>Автоматически переводит ответы на русский язык</li>
</ul>

<hr>

<h2>🧠 Функциональность</h2>

<ul>
  <li>Ввод пользовательского запроса (например: <em>"мне нравится кодить"</em> или <em>"люблю животных"</em>)</li>
  <li>Анализ запроса с помощью морфологического анализатора <strong>pymorphy2</strong></li>
  <li>Поиск подходящих профессий из базы данных <strong>SQLite</strong></li>
  <li>Вывод 1–3 релевантных профессий с описанием и навыками</li>
  <li>Команда <code>подробности</code> для получения дополнительной информации</li>
</ul>

<hr>

<h2>🛠️ Используемые технологии</h2>

<ul>
  <li><strong>Python</strong> — основной язык разработки</li>
  <li><strong>GPT4All</strong> — локальная LLM-модель для генерации текстов</li>
  <li><strong>SQLite</strong> — база данных для хранения профессий</li>
  <li><strong>pymorphy2</strong> — морфологический анализатор для работы с русским языком</li>
  <li><strong>translate</strong> — модуль для перевода ответов на русский язык</li>
</ul>
